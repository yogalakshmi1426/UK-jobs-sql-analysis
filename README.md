# UK Job Market Intelligence — SQL Analysis

> What does the data actually say about UK data analyst jobs? 
> I analysed 9,355 real job postings to find out.

## The Problem
Job seekers apply blindly without knowing which skills are in demand, 
what salary to expect, or which cities hire most. HR teams post jobs 
without knowing what competitors are offering. This project answers 
those questions with real data.

## What I Analysed
1. Which job titles appear most in the market?
2. Which experience levels are hiring and what do they pay?
3. Which locations have the most data roles?
4. What is the average salary by job title?
5. Remote vs Hybrid vs In-person — who pays more?
6. Which sectors are hiring most?
7. What is the salary journey from entry level to senior?
8. Is demand growing or shrinking over time?
9. Which company size hires most?
10. Full-time vs contract vs freelance — which pays more?
11. What does the UK market specifically look like?
12. Which categories have the most entry level roles?

## Key Findings
- Data job postings grew 10,392% from 2020 to 2023
- Data Analyst is the second most hired role in the UK with 85 postings
- Entry level Data Analyst averages $88,535 — jump to mid-level adds $29,000
- In-person roles pay more than remote — challenging common assumptions
- Medium sized companies hire the most and pay the best

## SQL Skills Demonstrated
SELECT, WHERE, GROUP BY, HAVING, ORDER BY, LIMIT,
AVG, COUNT, ROUND, DISTINCT, CASE WHEN, AS

## Dataset
Jobs in Data — Kaggle
9,355 real job postings across global data roles

## How to Run
1. Download jobs_in_data.csv from Kaggle
2. Load into PostgreSQL using the setup instructions
3. Run any query file from the queries folder

## Why I Built This
I built this project while actively job searching in the UK data market. 
I was asking myself these exact questions every day. 
Instead of guessing — I analysed the data.
